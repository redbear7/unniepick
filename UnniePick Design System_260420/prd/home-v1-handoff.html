01

왜
V1인가

3가지 홈 변형 중 V1을 Phase 1 단독으로 선정. V2·V3는 Phase 2 이후 A/B 테스트 대상으로 보류.

이유 01 — 구현 리스크

가장 적은 신규 컴포넌트

V1은 기존 FeedPost·StoreRow 자산을 80% 재사용. V2 히어로·카테고리 그리드·지도 미리보기는 모두 신규. V3 스토리 릴·스탬프 진행도 신규.

이유 02 — Spec 연결

Spec 01·02 연결점이 명확

긴급 쿠폰 스트립 = Spec 01 '지갑 발견'의 진입점. 내 주변 가게 카드 탭 = Spec 02 '스탬프 스캔'의 진입점. V1 화면 구조가 두 루프의 CTA 흐름과 1:1 매칭.

이유 03 — 학습 지표

섹션별 독립 측정 가능

5개 섹션이 수직 스택이라 각각의 스크롤 도달률·CTR을 독립 측정. 이후 V2/V3로의 리디자인 근거를 데이터로 확보.
