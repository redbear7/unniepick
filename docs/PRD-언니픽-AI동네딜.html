제품 한 줄 정의

언니픽은 “지금 내 상황에 맞는 동네 혜택”을 10초 안에 받게 하고, 관리자는 음악과 쿠폰을 결합한 숏폼으로 창원 상권을 유행시키는 앱이다.

사용자에게는 AI 로컬 커머스 어시스턴트, 사장님에게는 방문을 만드는 쿠폰 채널, 운영자에게는 음악 기반 숏폼 콘텐츠 공장을 제공한다. 새 사업을 별도로 만들기보다 현재 언니픽의 쿠폰 발행, 주변 가게, 피드, 지갑, 사장님 화면 위에 얹는다.

사용자 원칙

10초 안에 쿠폰 받기

상권 선택, 가격 선택, 쿠폰 받기까지 설명 없이 끝난다. 회원가입은 쿠폰 저장 이후 필요한 순간에 요청한다.

사장님 원칙

30초 안에 참여 신청

가게명, 대표 혜택, 사진 1장만 받는다. 복잡한 숏폼 편집과 카피 작성은 운영자가 대신한다.

디자인 원칙

da24식 미니멀 플로우

모바일 1열 화면, 큰 CTA, 하단 고정 버튼, 적은 선택지로 구성한다. 첫 화면은 예쁜 설명보다 즉시 행동을 우선한다.

참여 촉진 장치: 실시간 활동 피드 + 지오펜스 GPS

앱 첫 화면에는 “방금 상남동에서 쿠폰 3장 저장”, “모카팩토리 사장님이 라떼 1+1 등록”, “근처 300m 안에 오늘 쓸 쿠폰 4개”처럼 사용자와 사장님의 활동을 실시간 신호로 보여준다. 단, 개인의 실명, 전화번호, 정확한 좌표는 노출하지 않고 상권/반경 단위의 익명화된 활동으로만 표현한다.

창원 상권 활성화 버전

포지셔닝: “창원 만원픽” 또는 “창원 착한픽”

거지맵의 바이럴 문법은 가격이지만, 언니픽의 목적은 가격 후려치기가 아니라 창원 소상공인에게 실제 방문을 만들어주는 것이다. 따라서 외부 커뮤니케이션에서는 “거지”라는 자조적 단어보다 “만원으로 창원 한 끼”, “오늘 동네를 살리는 착한딜”, “창원 사장님 응원 지도” 같은 언어를 쓴다. 사용자는 절약의 재미를 얻고, 사장님은 신규 손님과 재방문 데이터를 얻는 구조로 설계한다.

소비자 절약 소상공인 방문 유도 창원 지역 캠페인 사용자 제보

거지맵에서 가져올 확산 장치

  • 1만원 이하, 7천원 이하 같은 즉시 이해되는 가격 필터
  • 사용자 제보로 빠르게 채워지는 지도
  • 점심값, 혼밥, 학생식, 직장인 밥값 같은 생활형 키워드
  • 커뮤니티 공유가 쉬운 짧은 이름과 캡처 가능한 카드
  • “지금 누가 쓰고 있다”는 실시간 활동감으로 참여를 유도

언니픽이 다르게 해야 할 점

  • 가게를 “싼 집”으로 낙인찍지 않고 “착한 구성”, “오늘 특가”, “사장님 응원딜”로 표현
  • 가격만 노출하지 않고 대표 메뉴, 마진 보호 조건, 방문 유도 혜택을 함께 표시
  • 사장님이 직접 정정, 인증, 쿠폰 발행할 수 있는 소유권 제공
  • 단순 절약 커뮤니티가 아니라 방문 인증과 재방문 쿠폰으로 연결
  • 거친 유머보다 깔끔한 카드와 숏폼 템플릿으로 공유 품질을 높임
  • GPS는 개인 추적이 아니라 상권 진입, 근처 혜택, 쿠폰 사용 가능성 판단에만 사용

창원에서 먼저 잡을 상권

  • 상남동: 직장인 점심, 저녁 술집 전환 쿠폰
  • 용호동/중앙동: 카페, 뷰티, 공공기관 주변 점심
  • 창원대/경남대 생활권: 학생식, 혼밥, 카페 공부석
  • 마산 어시장/창동: 전통상권 방문 챌린지
  • 진해 석동/여좌천: 주말 나들이와 디저트 동선

창원 바이럴 루프

단계 사용자 행동 소상공인 이득 제품 장치
발견 “창원 만원 이하 점심” 지도를 본다. 기존 광고 없이 신규 노출을 얻는다. 상권별 가격 필터, 오늘의 착한딜, 지도 캡처 공유
참여 저렴한 메뉴나 좋은 구성을 제보한다. 가게 정보가 무료로 보강된다. 30초 제보, 영수증/메뉴판 사진 첨부, 제보자 배지
인증 방문 후 가격/메뉴를 확인하고 후기를 남긴다. 실제 방문 증거와 신뢰 리뷰가 쌓인다. 방문 인증 스탬프, 가격 변경 신고, 사장님 답글
전환 쿠폰을 받고 재방문한다. 단발성 저가 방문이 재방문으로 바뀐다. 첫 방문 쿠폰, 2회차 스탬프, 한산 시간대 쿠폰
확산 “오늘 창원에서 만원으로 성공” 카드를 공유한다. 상권 단위의 유입이 늘어난다. 공유 카드, 주간 랭킹, 동네별 챌린지

소상공인 보호 원칙

  • 최저가 경쟁을 유도하지 않는다. “만원 이하” 외에도 “든든함”, “정직한 구성”, “재방문 혜택”을 같이 평가한다.
  • 사장님이 원치 않는 정보는 숨김/수정 요청할 수 있게 한다.
  • 가격 제보는 사용자 단독 확정이 아니라 사진, 영수증, 사장님 인증으로 신뢰도를 나눈다.
  • 쿠폰은 마진이 무너지는 상시 할인보다 한산 시간, 재고 소진, 첫 방문, 세트 업셀 중심으로 제안한다.

초기 캠페인 이름 후보

  • 창원 만원픽: 가장 직관적이다. 점심/카페/혼밥으로 확장하기 좋다.
  • 창원 착한픽: 소상공인 상생 이미지가 강하다. 지자체/상인회 협업에 유리하다.
  • 오늘창원딜: 음식 외 뷰티, 카페, 체험권까지 확장하기 쉽다.
  • 창원 사장님 응원딜: 바이럴성은 약하지만 상권 활성화 캠페인 메시지가 명확하다.

킬러 콘텐츠: 음악 쿠폰 숏폼 생성기

/dashboard/shorts는 창원 만원픽을 유행시키는 콘텐츠 엔진이다.

현재 개발 중인 관리자 숏폼 생성기는 언니픽이 보유한 음악 트랙과 활성 쿠폰을 결합해 9:16 MP4를 만든다. 이 기능을 사장님에게 바로 열기보다, 초기에는 운영자가 창원 상권별 대표 콘텐츠를 직접 제작해 품질을 통제한다. “상남동 점심 만원픽”, “창원대 카페 공부픽”, “마산 창동 사장님 응원딜” 같은 반복 가능한 시리즈를 만들어 앱 유입과 쿠폰 사용을 동시에 만든다.

관리자 제작 음악 + 쿠폰 Remotion MP4 릴스/쇼츠 배포

현재 구현된 핵심 기능

  • 관리자 음악 트랙 검색/선택: 제목, 아티스트, 태그 기반
  • 오디오 파형 표시와 구간 드래그 선택
  • 클라이맥스 자동 감지: /api/shorts/analyze
  • 영상 길이 선택: 10, 15, 20, 25, 30초
  • 활성 쿠폰 첨부와 영상 하단 쿠폰 카드 표시
  • Remotion 서버 렌더링: /api/shorts/render

콘텐츠 표현 요소

  • 숏폼 제목과 강조 문구
  • 트랙 커버 이미지 변경 또는 배경 동영상 업로드
  • 파형 디자인: Bar, Mirror, Wave, Circle, Dots
  • 커버 애니메이션: 호흡, 비트, 바이닐
  • 파티클 효과: 벚꽃, 비누방울, 하트, 별빛, 장미, 눈꽃
  • 제목, 쿠폰, 곡 정보, 파형 위치 조정

배포와 재사용

  • 렌더링 완료 후 MP4를 Supabase Storage에 업로드
  • 다운로드와 새 탭 열기 제공
  • 릴스 캡션 복사: 음악, 장르 태그, 언니픽 해시태그
  • 모바일 Web Share API로 인스타/틱톡 공유 가능
  • 로컬 히스토리에서 이전 설정 복원
  • 트랙별 생성 영상 히스토리 확인

창원 캠페인 시리즈 포맷

시리즈 영상 콘셉트 쿠폰/상권 연결 목표 행동
창원 만원 점심 15초 빠른 비트, 메뉴 사진, 가격 큰 자막, 쿠폰 카드 상남동, 중앙동, 창원대 앞 식당 점심시간 전 쿠폰 저장과 길찾기
퇴근 후 사장님 응원딜 저녁 무드 음악, 한산 시간대 방문 제안 술집, 카페, 네일, 마사지 한산 시간대 방문 분산
창원대/경남대 학생픽 짧고 반복적인 챌린지형 음악, 학생식/카페 쿠폰 대학가 혼밥, 카페 공부석, 디저트 제보 참여와 친구 공유
마산 창동 다시 걷기 로컬 감성 음악, 골목/시장 컷, 착한가격 인증 창동, 어시장, 전통상권 주말 방문 코스 저장

MVP 운영 방식

  • 초기에는 관리자만 숏폼을 만든다. 사장님 화면에는 “언니픽이 홍보 영상 제작 중” 상태만 보여준다.
  • 상권별 주 3편, 총 주 15편을 목표로 제작한다.
  • 각 영상은 반드시 하나의 행동만 유도한다: 쿠폰 받기, 길찾기, 제보하기, 공유하기 중 1개.
  • 영상에 들어가는 쿠폰은 사장님 인증 또는 운영자 검수된 것만 사용한다.

주의할 점

  • 음악 저작권과 배포 권한이 명확한 관리자 제작/보유 트랙만 사용한다.
  • 가게 사진, 메뉴 가격, 쿠폰 조건이 실제와 다르면 즉시 수정/비공개 처리한다.
  • 사장님에게 과도한 할인 압박처럼 보이지 않도록 “응원딜”, “한산 시간 혜택”, “첫 방문 혜택” 중심으로 표현한다.
  • 렌더링은 CPU 비용이 크므로 대량 자동 생성보다 승인형 큐와 템플릿 재사용을 우선한다.

MVP 범위

이번 버전에 반드시 넣을 것

  • 홈 상단 AI 질문 입력: “오늘 점심 1만원 이하”, “비 오는 날 카페”, “퇴근 후 네일 할인”
  • AI 추천 결과 5개: 거리, 시간대, 날씨, 카테고리, 쿠폰 조건 기반
  • 창원 상권 필터: 상남동, 용호동, 창원대, 마산 창동, 진해 석동 등 초기 지역 선택
  • 만원픽 가격 필터: 7천원 이하, 1만원 이하, 1만5천원 이하, 2인 세트 등
  • 사용자 제보 폼: 가게명, 메뉴명, 가격, 사진, 위치를 30초 안에 등록
  • 사장님 인증/정정 요청: 제보된 메뉴와 가격을 사장님이 직접 확인하고 쿠폰으로 전환

AI 기능 명세

A. 고객용 AI 추천

LLM은 “말을 해석하는 역할”까지만 맡기고, 실제 정렬은 기존 DB 점수식으로 처리한다.

  • 입력 예: “상남동 1만원 이하 점심”, “창원대 앞 카페 공부하기 좋은 곳”, “마산 창동 오늘 마감 쿠폰”
  • 출력 JSON: category, price_hint, time_context, mood, keywords
  • 랭킹 점수: 거리 35%, 시간대 20%, 만료/수량 20%, 카테고리 15%, 인기도 10%
  • 실패 시: AI 없이 추천 칩 기반 필터로 동일 결과 제공

B. 관리자용 숏폼 카피 생성

영상 렌더링은 /dashboard/shorts가 담당하고, AI는 숏폼 제목, 강조 문구, 캡션, 해시태그를 만든다.

  • 입력: 상권명, 가게명, 업종, 쿠폰명, 할인율, 음악 무드, 영상 시리즈명
  • 출력: 숏폼 제목 5개, 강조 문구 5개, 릴스 캡션, 해시태그, 앱 푸시 문구
  • 톤: 만원픽, 사장님 응원딜, 학생픽, 데이트픽, 한산 시간 혜택 중 선택
  • 저장: 선택된 문구를 숏폼 캠페인 메타데이터와 쿠폰 메타데이터에 같이 저장

1인 개발자용 우선순위 결론

가장 먼저 만들 기능은 “AI 추천 검색”이 아니라 “음악 쿠폰 숏폼으로 반복 노출되는 창원 만원픽”이다.

초기 사용자는 모델의 지능보다 “창원에서 지금 쓸 수 있는 혜택을 재미있는 영상으로 계속 보게 되는 경험”에 반응한다. 따라서 1차 구현은 관리자 숏폼 생성기를 캠페인 운영에 붙이고, 규칙 기반 추천 점수와 추천 이유 표시를 결합한다. 사장님 기능은 복잡한 편집기를 열기보다 쿠폰과 소재 제공, 인증/수정, 제작 동의 흐름에 집중한다.
