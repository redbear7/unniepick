배경 — 거지맵이 왜 100만을 모았나

🔥 거지맵 핵심 지표 (2026.03 출시)

출시 3주 만에 누적 94만 명 방문. 앱 설치·회원가입 없이 웹으로 즉시 접속. 핵심은 단 하나 — 1만 원 이하 식당을 지도에서 바로 찾는다. 고물가에 지친 2030 직장인이 "점심값 아꼈다"는 경험을 SNS에 공유하며 바이럴.

왜 지금인가?
언니픽은 이미 창원 맛집 DB·지도·리뷰를 보유하고 있습니다. 여기에 가격 필터 + 가성비 제보 보상만 추가하면 거지맵이 전국 단위로 못 하는 "창원 특화 초밀도 가성비맵"이 됩니다. 거지맵의 전국 DB가 창원을 채우기 전에 선점해야 합니다.

타겟 사용자
👩‍💼
Primary · 창원 직장인 (25~35세)
점심시간 12시에 지도를 켜서 도보 5분 이내, 8천원 이하 식당을 바로 찾고 싶다. 매일 같은 가게만 가기 싫고, 새로운 가성비 맛집을 발견하면 동료에게 공유한다.
🎓
Secondary · 창원대·경남대 대학생
한 달 식비 예산이 빠듯하다. 5천원 이하 식당을 미리 파악해두고, 가격 제보하면 코인을 받아서 쿠폰으로 교환하고 싶다.
🏪
Tertiary · 창원 소상공인 (가맹점주)
내 가게가 가성비맵에 노출되면 새 고객 유입이 생긴다. 가성비 뱃지를 받으면 앱 메인에 노출 기회가 생긴다.

DB 스키마 변경 (최소)
-- restaurants 테이블에 컬럼 추가
ALTER TABLE restaurants
  ADD COLUMN IF NOT EXISTS cheapest_price   integer,   -- 최저 메뉴 가격 (원)
  ADD COLUMN IF NOT EXISTS value_score      integer DEFAULT 0,  -- 올림 - 내림
  ADD COLUMN IF NOT EXISTS value_badge      boolean DEFAULT false; -- 가성비 뱃지

-- 가격 제보 테이블 (신규)
CREATE TABLE price_reports (
  id            uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  restaurant_id uuid REFERENCES restaurants(id),
  user_id       uuid REFERENCES users(id),
  menu_name     text NOT NULL,
  price         integer NOT NULL,
  status        text DEFAULT 'pending',  -- pending | approved | rejected
  created_at    timestamptz DEFAULT now()
);

-- 가성비 평가 테이블 (신규)
CREATE TABLE value_votes (
  id            uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  restaurant_id uuid REFERENCES restaurants(id),
  user_id       uuid REFERENCES users(id),
  vote          smallint NOT NULL CHECK (vote IN (1, -1)),
  created_at    timestamptz DEFAULT now(),
  UNIQUE (restaurant_id, user_id)
);

핵심 사용자 플로우
플로우 A — 가성비 점심 찾기
홈 탭 → 지도 탭 진입 → 상단 "💸 가성비맵" 토글 ON → 가격 슬라이더 8천원으로 설정 → 현재 위치 기준 핀 필터링 → 매장 카드 탭 → 메뉴·가격 확인 → 쿠폰 발급 → 방문
플로우 B — 가격 제보로 코인 획득
매장 상세 → "💬 가격 제보" 탭 → 메뉴명 + 가격 입력 → 제출 → 코인 10개 즉시 지급 알림 → 어드민 승인 (24시간 이내) → 지도에 반영
플로우 C — 가성비 평가
매장 카드 → 👍 올림 탭 → 로그인 확인 → 1일 1회 투표 반영 → 올림 10개 도달 시 자동 뱃지 부여 → 홈 TOP 10 노출

릴리즈 플랜
Phase 1 — 2주 이내
가성비맵 MVP
  • DB 스키마 추가 (cheapest_price, value_score, value_badge)
  • 지도 탭 가성비 필터 토글 + 가격 슬라이더
  • 매장 카드에 가성비 뱃지 + 최저가 표시
  • 올림/내림 투표 버튼
  • 홈 "이번 주 가성비 TOP 5" 섹션
  • 어드민 웹에 가격제보 승인 화면
Phase 2 — 런칭 후 2주
제보 보상 + 데이터 축적
  • 가격 제보 화면 + 코인 10개 보상
  • 가성비 랭킹 주간 자동 갱신 (크론)
  • 가격대 검색 필터 (예산 입력)
  • 푸시 알림 "내 동네 새 가성비 맛집 등록됨"
Phase 3 — 런칭 후 1개월
바이럴 + 가맹점 연동
  • 가성비 인증 공유 카드 (인스타 스토리)
  • 가성비 뱃지 매장 — 첫 방문 자동 쿠폰
  • 가맹점주 어드민에 가성비맵 노출 통계

가정 & 리스크
리스크 가능성 대응
거지맵이 창원 DB를 빠르게 채움 중 스탬프·쿠폰 연동 차별화. 거지맵엔 없는 로열티 시스템.
허위 가격 제보 어뷰징 중 어드민 승인 필수 + 올림/내림 자정 시스템
가맹점주 반발 (가격 공개 부담) 저 opt-in 방식. 참여 매장만 가성비맵 노출.
초기 데이터 부족 (가격 미입력 매장) 고 런칭 전 기존 DB에서 menu_items 가격 자동 추출. 부족분은 제보로 채움.

스탬프 & 쿠폰

창원 초밀도 DB
이미 수백 개 창원 맛집 DB 보유. 거지맵이 창원을 채우려면 시간이 걸림.
🏪
가맹점 네트워크
실제 가맹점주와 관계. 정확한 가격 정보를 공식 채널로 받을 수 있음.
🪙
코인 보상 생태계
제보하면 코인 → 쿠폰 교환. 거지맵은 무보상 자원봉사 구조.
